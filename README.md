## Установка

1. В корне переименовать файл .env.example в .env
2. Создать базу данных и внести необходимые правки в файл .env данные по субд
3. Выполнить команду php artisan key:generate
4. Выполнить команду php artisan migrate
5. Запустить локальный сервер командой php artisan serve
